# CG-Scale
Arduino-based aircraft center of gravity measurement system using HX711 load cells.
# Aircraft CG Measurement System

## Overview
This project measures the center of gravity of aircraft models using four load cells and an Arduino.

## Components
- Arduino Uno
- HX711
- 4 Load Cells
- Stainless Steel Frame

## Features
- Measures CG along one axis
- Displays weight readings
- Portable design

## Future Improvements
- Better wiring
- LCD display
- Battery-powered version
